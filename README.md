# TDE_change_detection

## Dataset

O projeto utiliza o dataset LEVIR-CD.

Estrutura esperada:

LEVIR_CD/
├── train/
│   ├── A/       # imagem pré-mudança
│   ├── B/       # imagem pós-mudança
│   └── label/   # máscara de mudança
│
├── val/
└── test/

O dataset pode ser obtido em:

https://www.kaggle.com/datasets/mdrifaturrahman33/levir-cd

Após baixar o dataset, coloque a pasta LEVIR_CD na raiz do projeto.
